# MathParadox
Демонстрация математических парадоксов
**Парадокс Галилея**
Открытый Галилео Галилеем феномен демонстрирует противоречивые свойства бесконечных множеств. 
**Суть**: Краткая формулировка парадокса такова: натуральных чисел столько же, сколько их квадратов, то есть, количество элементов бесконечного множества 1, 2, 3, 4… равно количеству элементов бесконечного множества 1, 4, 9, 16…
Рассмотрим два множества:
    1. Множество всех натуральных чисел: N = {1, 2, 3, 4, 5, ...}
    2. Множество квадратов натуральных чисел: S = {1, 4, 9, 16, 25, ...}
Казалось бы, очевидно:
Квадратов меньше, чем всех чисел. Ведь квадраты — это лишь часть ("подмножество") всех чисел, и между ними есть "пропуски" (2, 3, 5, 6, 7, 8... не являются квадратами). Но Галилей показал, что можно установить взаимно однозначное соответствие (биекцию):
    • 1 ↔ 1
    • 2 ↔ 4
    • 3 ↔ 9
    • 4 ↔ 16
    • n ↔ n²
Каждому натуральному числу можно поставить в соответствие его квадрат, и наоборот. Пропусков в этой переписке нет!
**Парадокс**: Как часть (квадраты) может быть равна целому (всем числам)? Это противоречит здравому смыслу, выведенному из опыта с конечными множествами.
**Разрешение**: Это свойство бесконечных множеств. У них часть может быть равномощна целому. Это краеугольный камень теории множеств Георга Кантора.
